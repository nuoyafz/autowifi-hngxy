# 校园网自动登录工具

Windows 平台校园网自动登录工具，基于 PowerShell 开发，无需安装 Python 或其他依赖，开箱即用。

## 功能特性

- ✅ **一键登录** - 输入账号密码，自动完成校园网认证
- ✅ **开机自启** - 勾选开机自动登录，每次开机无需手动操作
- ✅ **自动重试** - 登录失败自动重试 3 次
- ✅ **状态通知** - 登录成功/失败显示系统通知
- ✅ **静默运行** - 开机自启时无黑窗弹出
- ✅ **账号自动补全** - 只需输入纯数字账号，系统自动添加后缀
- ✅ **运营商选择** - 支持校园网(本地)、移动、联通、电信四种运营商
- ✅ **网络等待** - 开机自启时智能等待网络连接，避免认证失败

## 使用方法

### 1. 运行程序



**方式：** 右键 `校园网自动登录.ps1`，选择「使用 PowerShell 运行」

### 2. 首次配置

首次运行会弹出配置界面：

1. 输入校园网账号（纯数字即可，如：`250741111`）
2. 选择运营商（校园网本地/移动/联通/电信）
3. 输入校园网密码
4. 勾选「开机自动登录」（可选）
5. 点击「保存并登录」

### 3. 日常使用

- **手动登录**：双击 `启动校园网.bat`，点击「立即登录」
- **自动登录**：开机后自动登录（需勾选开机自启）
- **管理自启**：点击「开机自启」可随时开启/关闭开机自动登录

## 常见问题

### Q: 开机后没有自动登录？
A: 请确认已勾选「开机自动登录」，同时确保开机后已连接校园 WiFi。脚本会自动等待网络连接（最多30秒）。

### Q: 登录失败怎么办？
A: 
1. 检查账号密码是否正确
2. 检查运营商选择是否正确
3. 检查是否已连接校园 WiFi
4. 查看 `login.log` 日志文件排查问题

### Q: 如何卸载/关闭开机自启？
A: 点击「开机自启」按钮，选择「关闭」即可。

### Q: 开机自启时认证失败？
A: 脚本已内置网络等待功能，会自动等待网络连接后再尝试登录。如果仍失败，请检查 WiFi 是否正常连接。

## 技术原理

- **语言**：PowerShell + Windows Forms
- **认证方式**：HTTP GET 请求校园网认证接口
- **自启方式**：Windows 开机启动项（Startup 文件夹快捷方式）
- **IP 获取**：通过 TCP 连接获取本机 IP 地址
- **网络检测**：使用 Get-NetConnectionProfile 检测网络状态
